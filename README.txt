
# AI Phishing Agent chat bot: This Agent is an AI powered Phishing Risk Detector
**Version 1.0.0** 


A Student-Friendly AI agent built using Streamlit that detects phishing risks. Available to all. 
#An additional feature of detecting phishing scams relating to Financial Aid, or FAFSA - making this useful for students.

## FEATURES

-Keyword detection system
- Scans for (Safe & Danger detection system) - contains risks scoring
- Safe word dampening to reduce the risk of false positives
-SHA-256 Hash function to maintain integrity of the word lists. 

**Color coded alerts**
Green - no risk, 
Yellow - low risk, 
orange - Medium risk
red - High risk


## Installation & Usage 

Before running PhishBot you may have to install additional requirements before using Agent Chat bot
run these commands in the Python Environment or Pycharm 

#Step 1:  Install dependencies

pip install -r requirements.txt 
 
#Step 2: Launch the app
 
streamlit run phishbot.py
